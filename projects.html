Fusion 360

Exercise 1 - Name Tag

For this exercise, we learnt how to create a 3D name tag using Fusion 360. In order to create this name tag, we used Sketch to create the outline of the name tag, Trim to trim off excess lines, adding Text and also Extrusion to pull out the surfaces.

Exercise 2 - Lego Brick

For this next exercise, we will have to create a lego brick that is hollow underneath. For this we learnt to use the Shell tool to create the hollow side of the lego brick. We also leanrt how to create 3D objects on top of surfaces and duplicating them, this creates the knobs of the brick.
