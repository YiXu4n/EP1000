Exercise 3 - Knight Chess Piece



Step 1 - Trace

To create this Knight Chess piece, i first download an image of a chess piece from Google and 'Insert' > 'Canvas' into Fusion 360 to trace out the outline of the piece.



Step 2 - Extrude (Head)

Next, to select surface and extrude out 5mm for thickness of the head.



Step 3 - Evolve (Base)

I will first sketch out the shape of the base and select elvolve to form the base of the chess.



Step 4 - Shell (Hollow Base)

Select the Shell tool and select the bottom surface will create a hollow base for the chess.



Step 5 - Preparing the files

The next thing i'll do is to convert the file to a .STL file and slice it in Cura. In Cura, i'll adjust the settings and make sure that overhanging areas are supported. Next, i will save it to disk and use a 3D Printer to produce the chess piece.

Using Fusion 360, i created a 3D model of the music box with all the finger taps. Next step, I save the sketches as a DXF file and open it in a 2D software (I use Illustrator) to edit and add some designs to it. When that is done, I will start laser cutting it out.


Place all the pieces down on a cutting mat as shown in the photo. To secure all the pieces together, you can use a wood glue if you have it but in this case I don't so I'll be using UHU glue. Get ready the music mechanism too.

Next step is to glue all the pieces together but with a sequence. Starting from glueing down the music mechanism to the base piece (A) of the music box.
